# ⚡ Smart Energy Consumption Prediction System
This project focuses on predicting household energy consumption using machine learning techniques. 
The dataset contains real-world electricity usage data, which is analyzed to identify patterns and trends.

objectives:-
- Predict energy consumption using ML models
- Compare performance of different algorithms
- Analyze energy usage patterns

Dataset: Household Power Consumption Dataset

Features Used:
- Voltage
- Global Intensity
- Hour
- Month

Target Variable:
- Global Active Power

Technologies used:-
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

- Linear Regression
- Decision Tree Regressor
- Random Forest Regressor

- Calculated RMSE for each model
- Compared model performance
- Random Forest performed best

Visualisation:-
- Daily energy consumption trend
- Energy usage by hour
- Energy usage by month
- Correlation heatmap
- Model comparison graph

to run:-
1. Install dependencies:
   pip install -r requirements.txt

2. Run the project:
   python solarprediction.py
